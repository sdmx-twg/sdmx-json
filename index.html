JSON schemas

This site publishes the official XSD schemas for the SDMX-JSON transmission format covering structural metadata, data and referential metadata. Schemas are provided for versions 1.0, 2.0.0 and 2.1.0 of the SDMX-JSON specification. The full specifications can be found on the GitHub repository which also includes documentation and samples.
